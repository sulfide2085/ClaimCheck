ClaimCheck / GitHub Pages Demo

让竞品分析从“看评论”升级为“用评论验证卖点”。

ClaimCheck 是一个面向跨境电商运营的 AI 小工具。它把竞品页面卖点、详情图表达与用户真实评论放到同一个判断链中，帮助团队快速判断哪些卖点值得学，哪些存在风险。

定位 卖点验证助手

不是做泛化评论总结，而是做更接近运营决策的一步判断。

输入 Listing + 评论 + 图片

标题、五点、详情页文案、详情图、好评和差评统一进入分析链。

输出 可学 / 慎学 / 不建议学

每条结论都要求带证据，避免只给一个空判断。

Pain Point

真正缺的不是再多一个评论分析工具，而是判断竞品卖点真假。

很多工具可以做情绪分析、关键词统计、差评归类，但运营在真正做决策时，最关心的问题其实是：竞品页面写的卖点，到底有没有被用户真实体验支撑？

Why Now

因为 LLM 已经能理解“承诺”和“体验”之间的对应关系。

用户不会总是原样复述页面文案，但会用真实场景描述体验。ClaimCheck 的价值就在于把这些间接证据翻译成运营能用的判断。
